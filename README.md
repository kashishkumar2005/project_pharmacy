Ef**ficient medicine inventory management** is crucial for pharmacies to ensure drug availability 
and minimize waste. Traditional manual systems are inefficient and prone to errors. This project 
is a web-based application designed to automate core pharmacy operations using modern 
technologies.

**Technology Stack:**

• Frontend: React.js (for a responsive and user-friendly UI)  
• Backend: Java + JDBC (for business logic and database operations)  
• Database: MySQL (for reliable data storage)
